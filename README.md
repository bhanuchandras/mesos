# mesos
Zero to Mesos on GCP with terraform and ansible.

This repor is designed to be get you started with mesos and marathon installed with following commands.
### Pre-requistes
* Terraform
* ansible

```
terraform init
terraform plan
terraform apply      # need to say yes to go ahead
ansible-playbook play-mesos.yaml
```

please let me know if you run into any issues, i will try to make this repo as generic as possible.
